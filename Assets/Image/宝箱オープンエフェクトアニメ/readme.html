宝箱オープンエフェクトアニメーション画像

●収録内容

宝箱などを開けたときのアイテム入手演出での使用を想定したエフェクトアニメです

アニメパターンをスプライトシート状に並べたPNG画像となっています

アニメーションの表示速度は15fpsで作成されています

ファイルサイズの最適化はしておりませんので、必要に応じて下記の「注意事項」で紹介しているアプリなどでファイルサイズを小さくすることをお奨めします

サンプルアニメ

「png」フォルダ

ファイル名 内容 サイズ展開 パターン数(並び) 推奨描画モード 備考
pipofm-topen01_.png アイテムゲット　小く 480×480,192×192 10(5×2) 通常,加算
pipofm-topen02_.png アイテムゲット　中く 480×480,192×192 15(5×3) 通常,加算
pipofm-topen03_.png アイテムゲット　大く 480×480,192×192 20(5×4) 通常,加算

●Unity用データ

実験的にUnity用のアセットデータを収録しています。

unity_assetフォルダの中にTreasureChestOpenEffect.unitypackageというパッケージファイルを入れてありますので、プロジェクトにインポートしてご利用ください。

なにか不具合などありましたらコメントなどでお知らせいただけると助かります。
